About Me

Hello! I'm Mary Adewumi, an enthusiastic and detail-oriented Frontend Developer based in Lagos, Nigeria. I’m passionate about turning creative ideas into functional and responsive websites that users enjoy.

I recently completed a tech bootcamp where I collaborated with a team to build real-life projects like a safety alert app (BeepSafe). These experiences helped me grow technically and strengthened my teamwork and communication skills.

I also took a course in Product Design, which helped me understand how to structure user-friendly interfaces. I know my way around Figma and can confidently use it to create wireframes, mockups, and UI prototypes. This design knowledge complements my frontend work and allows me to bring ideas to life from concept to code.

My core strengths include:

  • Writing clean, semantic HTML
  • Designing responsive layouts with CSS
  • Adding interactivity with JavaScript
  • Deploying live projects using GitHub & Vercel

I’m currently learning React.js. I enjoy solving problems, learning new tools, and continuously improving my skills.

Skills: HTML, CSS, JavaScript, GitHub, Vercel, React (beginner), Figma

In my free time, I enjoy exploring UI design ideas, watching web development tutorials, and helping others solve beginner coding challenges.
